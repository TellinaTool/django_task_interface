Instructions

Welcome to the bash commands user study. Thank you for participating!


1. Bash tasks

You will be given 18 tasks related to file system operations. The desired outcome of each task may be to

  • change the file system by deleting, adding, or modifying files, or
  • produce output, such as a list of files

    • Your goal is to write a bash one-liner that performs the desired operation without causing extra file system changes. You will work in a bash shell in your browser. You can also reset the file system to its original state at any time.

    You may attempt the task multiple times. Each task has a time limit of 5 minutes, after which you will proceed to the next task even if you have not yet completed the current task.


    2. Tool assistance

    For all tasks, you may use Internet resources such as Google search. For some tasks, you will be permitted to use additional tools.


    3. Questionnaire

    You will fill in a post-study questionnaire at the end of the study.
