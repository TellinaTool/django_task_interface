Instructions

Welcome to the Tellina user study! Please read the instructions below carefully.


1. About the Study

In this study, you will work on 18 tasks related to file system operations. The desired outcome of each task is either a list of files or a change in the file system, such as deleted, added or modified files. Your goal is to write a bash one-liner that performs the desired operation without causing extra file system changes.

You will complete the 18 tasks in two parts, each consists of 9 tasks. In part one, you may look up the relevant bash command using either Tellina or Google Search (including any resources you can find on the Internet). In part two, you may look up the relevant bash command using Goolge Search only.

You would need to complete a training task at the beginning of each part. The purpose of the training task is to get you familiar with the task platform and the assistant tool(s).

You would need to fill in a post-study questionaire when you completed the tasks in both parts.

2. About the Task

For each task, you can write and execute bash commands in a web shell, which will affect a connected file system. Each task has a time limit of 5 minutes.

Within the time limit, you may attempt a task multiple times. You also have the option to reset a file system in case it reached a non-recoverable state. There is no limit on the number of commands you may issue or the number of file system resets you can perform.

When you executed a bash command, the outcome is automatically verified and you will be redirect to the next task if the goal is achieved. You will also be redirected to the next task when times up, regardless of whether the current task is completed or not.
