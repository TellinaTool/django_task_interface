Instructions

Welcome to the bash commands user study. Thank you for participating!

For the best view of the task interface, please turn on the full-screen mode of your browser.

This page contains important information regarding the study session. Please read it through before you proceed.


1. Bash tasks

You will be given 18 tasks related to file system operations. Each task may require you to

  • change the file system by deleting, adding, or modifying files, or
  • produce output, such as a list of files

    • Your goal is to write a bash one-liner that performs the desired operation without causing extra file system changes.

    You will work in a bash shell in your browser and may attempt the task multiple times.


    2. Tool assistance

    {% if treatment_order == "0" %}

    For all tasks, you may use Internet resources such as Google search. And for the first half of the study (the first 9 tasks you encounter), you will be permitted to use additional tool(s).

    {% elif treatment_order == "1" %}

    For all tasks, you may use Internet resources such as Google search. And for the second half of the study (the last 9 tasks you encounter), you will be permitted to use additional tool(s).

    {% endif %}

    3. Timing

    Each task has a time limit of 10 minutes, after which you will proceed to the next task even if you have not yet completed the current task.

    In addition, you may spend maximally 40 minutes to complete each half of the study (9 tasks), after which you will proceed even if you have not yet completed the current half.


    You will first complete a task platform training before getting to the real tasks. The training session will help you get familiar with the task platform as well as the additional tool(s) that you will use.
